Install Google Places Autocomplete Add-On ¶

The Google Places Autocomplete Add-On installing process is very simple, same to any other Add-On. To install the Google Places Autocomplete Add-On follow the documentation to install an Add-On from the Add-On Manager.

Create an Address Autocomplete Field ¶

To create an Address Autocomplete Field, you must perform the following steps:

  1. Go to the Add-ons Manager and click on Google Places Autocomplete.
  2. Click the Set Up a Form button.
  3. A form used to configure the address autocomplete will appear.
    • Form: Select the form.
    • Api Key: Enter your Google Places Api key.
    • Status: Can be:
      • On: Enable autocomplete
      • Off: Disable autocomplete
    • Fields: It's composed of:
      • Field: Choose the field to be auto-completed.
      • Type: The type of search you want to use: "Address", "Establishment" or "All".
      • Country: Select a country to limit the search to an specific country.
  4. Click Create
