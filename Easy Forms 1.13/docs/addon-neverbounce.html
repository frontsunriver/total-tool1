Install NeverBounce Add-On ¶

The NeverBounce Add-On installing process is very simple, same to any other Add-On. To install the NeverBounce Add-On follow the documentation to install an Add-On from the Add-On Manager.

Create a NeverBounce Validation ¶

To create a NeverBounce's validation, you must perform the following steps:

  1. Go to the Add-ons Manager and click on NeverBounce.
  2. Click on the Set Up a Form button.
  3. A form used to configure the validation will appear.
    • Form: Select the form to validate.
    • Private Api Key: Enter the NeverBounce access credential.
    • Public Api Key: Enter the NeverBounce access credential.
    • Status: Can be:
      • On: Enable validation
      • Off: Disable validation
    • RealTime: Enable / Disable realtime validations.
    • Fields: It's composed of:
      • Field: Choose the field to be validated.
      • Field Message: The custom error message if the value entered by the user is an invalid email.
  4. Click Create
